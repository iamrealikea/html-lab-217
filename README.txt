การเรียนวันนี้แสดงให้เห็นถึงความเป็นมาของ Internet และ World-Wide-Web ที่เป็นจุดกำเนิดของโลกอินเทอร์เน็ตทั้ง frondend, backend และ database ที่พวกเราทุกคนใช้กันในชิวิตประจำวัน โดยมี html เป็นส่วนประกอบที่สำคัญในหน้า front-end รวมถึง CSS และ  JavaScript ในการตกแต่งหน้าเว็บให้ใช้งานง่ายและน่าใช้ ได้รู้ถึง markup พื้นฐานต่างๆในการเขียน html และได้ลองนำมาใช้เขียนจริงดังใน lab วันนี้
